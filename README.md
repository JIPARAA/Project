เพื่อช่วยแก้ปัญหาผู้ที่ต้องการออกกำลังกาย แต่ขาดความรู้ความเข้าใจในการประเมินสภาวะความพร้อมของร่างกายตัวเองในแต่ละวัน (เช่น การคำนวณค่าดัชนีมวลกาย (BMI), ระยะเวลาการพักผ่อน, และระดับความเหนื่อยล้าสะสม) ระบบนี้จะช่วยประเมินและให้คำแนะนำรูปแบบการออกกำลังกายที่ปลอดภัยและเหมาะสมกับสภาพร่างกาย ณ ขณะนั้น เพื่อป้องกันอาการบาดเจ็บและอันตรายจากการฝืนร่างกาย

# 🚀 ฟีเจอร์หลัก (Features)

1 ระบบคำนวณค่า BMI อัตโนมัติ
    - ผู้ใช้กรอกน้ำหนักและส่วนสูง ระบบจะคำนวณและจัดกลุ่มดัชนีมวลกายให้อัตโนมัติ (ผอม, ปกติ, ท้วม, อ้วน 1, อ้วน 2) พร้อมระบบป้องกันข้อผิดพลาดหากผู้ใช้กรอกค่าเป็น 0

2 ประมวลผลระยะเวลาการพักผ่อนอัจฉริยะ 
    - ผู้ใช้เพียงกรอกเวลาเข้านอนและเวลาตื่นนอน ระบบสามารถคำนวณชั่วโมงการนอนหลับสุทธิได้อย่างแม่นยำ (รองรับการคำนวณเวลาข้ามวัน) และจัดกลุ่มการนอนหลับ (น้อย, ปานกลาง, มาก)

3 ประเมินระดับความเหนื่อยล้า (Fatigue Level) 
    - ประเมินความรู้สึกผ่านการเลื่อนแถบระดับ (Slider 0-10) เพื่อความง่ายในการใช้งาน

4 ระบบผู้เชี่ยวชาญให้คำแนะนำ (Expert System) 
    - โหลดเงื่อนไขและข้อมูลคำแนะนำจากฐานข้อมูลไฟล์ Excel (project.xlsx) นำมาประมวลผลร่วมกับค่าที่คำนวณได้ เพื่อดึงคำแนะนำที่ตรงกับสภาวะร่างกายแบบ 100%

5 การจัดการข้อผิดพลาด (Error Handling) 
    - มีระบบตรวจสอบความถูกต้องของข้อมูล เช่น แจ้งเตือนเมื่อกรอกรูปแบบเวลาผิด (นาทีเกิน 59)หรือเมื่อรูปแบบชื่อคอลัมน์ในฐานข้อมูลไม่ตรงกัน ระบบจะแจ้งเตือนแทนการหยุดทำงาน (Crash)

# 👤 ผู้พัฒนาและภาระงาน (Developer & Responsibilities)

### นายเมฆินทร์ ไทยเจริญ 

Day 1: วางแผนโครงงาน รวบรวมทฤษฎี/เกณฑ์ประเมินด้านสุขภาพ ออกแบบเงื่อนไขคำแนะนำ และสร้างฐานข้อมูล Rule-base ในรูปแบบไฟล์ Excel

Day 2: พัฒนา Web Application สร้างโครงสร้างหน้าเว็บ เขียนฟังก์ชันรับค่าข้อมูลและคำนวณ (BMI, เวลาพักผ่อน) และเชื่อมต่อระบบเข้ากับฐานข้อมูล Pandas เพื่อดึงข้อมูลคำแนะนำ

Day 3: ตรวจสอบข้อผิดพลาด (Bug) สร้างระบบป้องกันการกรอกข้อมูลผิดพลาด นำระบบขึ้นออนไลน์ผ่าน GitHub และ Streamlit Cloud รวมถึงทดสอบการใช้งานจริงและจัดทำเอกสารสรุปผล

- Libraries: * pandas และ openpyxl (สำหรับอ่านและจัดการข้อมูลในไฟล์ Excel)
